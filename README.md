Clawd Listen is a native macOS floating panel that transcribes microphone and system audio in parallel, then lets you query the live transcript and your local knowledge base through Claude. Built for meetings where you need a silent co-pilot — not another bot in the call.

## Features

- **Dual-stream transcription** — mic and system audio captured independently, streamed to OpenAI `gpt-4o-transcribe` with speaker labels.
- **In-meeting Q&A** — query the live transcript with Claude, grounded in a user-defined knowledge folder (e.g. Obsidian vault) via `Glob` / `Grep` / `Read`.
- **Subscription-aware** — Q&A routes through `claude -p`, reusing your Claude Max plan at no extra cost.
- **Floating panel** — always-on-top, resizable, hidden from screen sharing (stealth window).
- **Auto-export** — meeting title generated by Claude on stop, saved as `YYYY-MM-DD_HH-mm_<title>.md`.
- **Echo cancellation** — Voice Processing (AEC + AGC + noise suppression) prevents double-capture in online calls.
- **Low overhead** — throttled VU metering and permission checks for all-day use.

## Requirements

| | |
|---|---|
| OS | macOS 14 (Sonoma) or later |
| OpenAI | API key with access to `gpt-4o-transcribe` |
| Claude Code | `claude` CLI on `$PATH` (Pro or Max subscription recommended) |
| Permissions | Microphone, Screen Recording |

Transcription cost: ~$0.7 per hour of meeting (two parallel streams).

## Install

Download the latest DMG from [Releases](https://github.com/yuyuyupoke/clawd-listen/releases), then:

```bash
# First launch only — bypass Gatekeeper (ad-hoc signed)
xattr -cr "/Applications/Clawd Listen.app"
open "/Applications/Clawd Listen.app"
```

Or build from source:

```bash
git clone https://github.com/yuyuyupoke/clawd-listen.git
cd clawd-listen
./scripts/setup-signing.sh
./build.sh
```

## Usage

Set the OpenAI key, output folder, and (optionally) a knowledge folder on first launch — credentials are stored in Keychain.

**Record a meeting**

Click the record button. Mic and system audio stream in parallel, labeled `[self]` and `[other]`. Click stop to auto-title and export Markdown to your output folder.

**Ask during a meeting**

Type a question in the right panel. Claude answers using the live transcript, your knowledge folder, and web search.

| Question | Behavior |
|---|---|
| "What's this term mean?" | Summarized from live context |
| "Find related notes" | Grep/Read across knowledge folder |
| "Latest industry data?" | WebSearch + WebFetch |

**Run as a background agent**

```bash
sed "s|\$HOME|$HOME|g" config/com.clawdlisten.agent.plist \
  > ~/Library/LaunchAgents/com.clawdlisten.agent.plist
launchctl bootstrap "gui/$(id -u)" ~/Library/LaunchAgents/com.clawdlisten.agent.plist
```

## Architecture

```
┌──────────────────────────────────────────────────────────────┐
│  Clawd Listen (Swift / SwiftUI)                              │
├──────────────────────────────────────────────────────────────┤
│  ┌──────────────┐   ┌────────────────────┐   ┌────────────┐  │
│  │ AVAudioEngine│   │  ScreenCaptureKit  │   │ AppKit UI  │  │
│  │     mic      │   │   system audio     │   │ float panel│  │
│  └──────┬───────┘   └─────────┬──────────┘   └─────┬──────┘  │
│         │  PCM 24kHz mono     │                    │         │
│         ▼                     ▼                    │         │
│  ┌──────────────────────────────────┐              │         │
│  │ OpenAI Realtime API (WebSocket)  │              │         │
│  │  gpt-4o-transcribe · server_vad  │              │         │
│  └────────────────┬─────────────────┘              │         │
│                   │ delta / completed              │         │
│                   ▼                                │         │
│  ┌─────────────────────────────┐                   │         │
│  │ TranscriptStore             │ ──────────────────┤         │
│  └──────┬──────────────────────┘                   │         │
│         │                          ┌───────────────┴──────┐  │
│         ▼                          │ Q&A input            │  │
│  ┌─────────────────────┐           ▼                      │  │
│  │ TranscriptExporter  │     ┌─────────────────────┐      │  │
│  │   Markdown out      │     │ ClaudeQAClient      │ ◀────┤  │
│  └─────────────────────┘     │  claude -p · --add-dir│    │  │
│                              └─────────────────────┘      │  │
└──────────────────────────────────────────────────────────────┘
```

| Source | Responsibility |
|---|---|
| `AudioSession.swift` | Session lifecycle |
| `MicrophoneCapture.swift` | `AVAudioEngine` + Voice Processing |
| `SystemAudioCapture.swift` | `ScreenCaptureKit` tap |
| `TranscriptionClient.swift` | OpenAI Realtime WebSocket |
| `ClaudeQAClient.swift` | Claude CLI subprocess + prompt assembly |
| `TranscriptExporter.swift` | Markdown rendering |
| `FloatingPanel.swift` | `NSWindow` floating panel |
